**PERFEKT**

Das Perfekt besteht aus zwei Teilen:

Was haben Sie gestern gemacht?

| 1. Hilfsverb | 2. Partizip II | 
| :----- | :----- |
| Ich **bin** | mit dem Auto **gefahren** |
| Ich **habe** | das Problem **gelöst** |

**Perfekt mit sein oder haben**

| sein | haben 
|:--- | :--- |
| ich **bin** gefahren |ich **habe** getrunken |
| ich **bin** aufgestanden | ich **habe** gearbeitet |
| ich **bin** gewesen | ich **habe** gefrühstückt |
| ich **bin** geblieben | ich **habe** geschrieben |


**Das Partizip II**


**Verben ohne Präfix** 

| regelmäßige Verben | unregelmäßige Verben | Verben mit -ieren |
| :----- | :----- | :----- |
| fragen -> **ge**frag**t** | trinken -> **ge**trunk**en**| studieren -> studier**t** |
| arbeiten -> **ge**arbeite**t** |sprechen -> **ge**sproch**en** | kopieren -> kopier**t** | 


**Verben mit Präfix** 

| trennbare Verben | nicht trennbare Verben |
| :----- | :----- |
| **regelmäßige Verben** | **regelmäßige Verben** |
| einkaufen -> ein**ge**kauf**t** | besuchen -> besuch**t** |

| trennbare Verben | nicht trennbare Verben |
| :----- | :----- |
| **unregelmäßige Verben** | **unregelmäßige Verben**|
| anrufen -> an**ge**ruf**en** | beginnen -> begonn**en** |
