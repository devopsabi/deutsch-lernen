**Past tense**

German past tense primarily uses Perfekt (conversational) for spoken launguage, formed with haben/sein + past participle, and Präteritum (written) for narratives,using specific verb stem changes.


| Present Tense | Past Tense |
| :----- | :----- |
| I Play | I have played |
| ich spiele | Ich habe gespielt |

**The Helping Verb**

| If in the main verb there is a movement or transition of state | No Movement |
| :----- | :----- |
| ich bin | ich habe |
| du bist | du hast |
| er/es/sie ist | hat | 
| wir sind | wir haben |
| ihr seid | ihr habt |
| sie/Sie sind | sie/Sie haben |


**The Past Participle**

| Weak Verbs (regular) | Strong Verbs (irregular) |
| :----- | :----- | 
| gespielen - getspielt | gesprochen - **keeps the** en |
| gefragen - gefragt | gesehen |
| Kennen - gekannt | gekommen | 
| | gegangen |



**Trick for figuring out the past participle**

| If the english workd has en or t/ed the German will also have en or t/ed |
| :-- | :-- |
| English | German | 
|  ... taken | Ich habe genommen |
| ... spoken | Ich habe gesprochen |
| ... eaten | Ich habe gegessen |
| ... written | Ich habe geschrieben | 
| ... brought | Ich habe gebracht |
| ... cooked | Ich habe gekocht |
| ... learned/ed | Ich habe gelernt |
| ... heard | Ich habe gehört |


**Mixed Sentences - 2 different past tense forms**

I went home and studied

Ich bin nach Hause gegangen und habe gelernt.

**Notice:** bin is used for 1st verb going. We use habe for the 2nd verb because studying is an activity that does not require signficant movement. You need to make sure that you change the helping word in the sentence.

I drove home and went hiking.

Ich bin nach Hause gefahren und wandern gegangen.

**Notice:** We only need one helping word bin because both verb are type of movement and they dont need second helping verb.
