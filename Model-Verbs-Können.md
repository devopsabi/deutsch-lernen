The German verb können (can/ to be able to) is an irregular modal verb.

In the present tense, it changes vowel (ö to a) for singular forms.

Where ich/er/es/sie use "kann"

Plural forms retain the "ö".

It uses haben as an auxiliary in perfect tenses. (e.g., ich habe gekonnt). 


***Present Tense (Präsens)***

* ich kann - I can

* du kannst - you can

* er/es/si kann - he/it/she can

* Wir können - We can

* Ihr könnt - You all can

* sie/Sie können - (They/You can - Formal)


***Past Tense (Präteritum)***

* ich/er/es/sie konnte (I/he/it/she could)

* du konnstest ( you could)

* Wir/sie/Sie konnten - (we/they/you could)

* ihr konntet ( you all could )

***Perfect Tense (Perfekt)***

* ich habe gekonnt - I have been able to/could

--------------------------------------------------------------------------------------------------------------------

***sub + modal + ----- infinitive verb.***


ich kann Deutsch sprechen. - I can speak German.

ich kann Yoga machen. - I can do Yoga.

ich kann unterrichten. - I can teach

Ich kann heute nicht nach Hause gehen. - I cannot go home today.

Kann ich deinen Kuli nehmen? - Can I use your pen?

Wer kann das machen? - Who can do that?
