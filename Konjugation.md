1. Präsens (Present Tense)

* Used for action happening now or regularly.

Regular verbs pattern.

Stem + endings:

-   ich -e
-   du  -st
-   er/es/sie -t
-   wir -en
-   ihr -t
-   sie/Sie -n

Beispiel: machen (to do)

| Subjekt| konjugation|
|:---|:---|
| ich | mache |
| du | machst |
| er/es/sie | macht |
| wir | machen |
| ihr | macht |
| sie/Sie | machen |

Beispiel:

-   Ich **lerne** Deutsche.
-   Er **arbeite** viel.

2. Präteritum (Simple Past)

Used often in writing (stories, reports, news).

Regular verbs add **-te** to the stem.
Example: machen

| Subjekt | konjugation |
| :--- | :--- |
| ich | machte |
| du | machtest |
| er/es/sie | machte |
| wir | machten |
| ihr | macttet |
| sie/Sie| machten |


Irregular verbs: stem changes and use different endings.
Example: gehen (to go)

| Subjekt | konjugation |
| :--- | :--- |
| ich | ging |
| du | gingst |
| er/es/sie | ging |
| wir | gingen |
| ihr | gingt
| sie/Sie | gingen |


Beispiel:

-   Ich **machte** Hausaufgaben.
-   Er ging nach Hause.

3. Perfekt (Conversational Past)

Used more often in spoken German.

Formed with haben or sein + Partizip II.

Regular verbs:
-   Participle = **ge + stem + t**
    Beispiel: machen -> gemacht

Irregular Verbs:
-   Participle ends in -en, often with vowel 
    Beispiel: gehen -> gegangen

**haben** or **sein**

-   haben -> for most verbs
-   sein -> for movement or state-change verbs (gehen, kommen, bleiben, werden)

Beispiel:
    -   Ich habe Deutsch gelernt.
    -   Ich bin nach Berlin gegangen
    -   Sie hat Koffee getrunken
